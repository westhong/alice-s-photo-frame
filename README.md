# ✨ Alice's Photo Frame

A magical photo frame web app for Alice — add cute white frames and "photo by Alice" watermarks to your photos!

## Features
- 📸 Upload any photo from your phone or device
- 🖼️ Automatically adds a pretty white frame
- ✍️ Adds "photo by Alice" watermark
- 💾 Save the framed photo back to your device
- 📱 PWA — works offline, installable on phone

## Tech Stack
- Pure HTML/CSS/JavaScript (no frameworks needed!)
- Cloudflare Workers + Static Assets
- GitHub Actions for CI/CD

## Deployment
Deployed via Cloudflare Workers. Set these GitHub secrets to enable auto-deploy:
- `CLOUDFLARE_API_TOKEN` — your Cloudflare API token
- `CLOUDFLARE_ACCOUNT_ID` — your Cloudflare account ID

## Development
Just open `index.html` in a browser — no build step required!
